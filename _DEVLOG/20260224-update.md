# Measure Canvas rendering performance
- Use FPS and GPU load as a measurement

### GPU load measurement
- Chrome DevTools - GPU Monitor (Most Direct)
```
- Open DevTools → Performance → Rendering tab
- Enable "Paint timing" and "GPU tasks"
- Record while interacting with canvas
- See actual GPU time per frame in milliseconds
```

#### GPU load - Practical Interpretation
|AvgFrame Time|	Headroom|	Can Add more?|
|--|--|--|
|< 8ms|	> 50%|	✅ Heavy elements|
|8-12ms|	25-50%|	⚠️ Moderate elements|
|12-15ms|	< 25%|	❌ Limited capacity|
|> 16.67ms|	Negative|	🔴 Already dropping frames|
